Hedonism

不管在我活着的哪个阶段，电子游戏总是充斥于我的同学朋友间，他们应该都没抱有任何目的，只是去玩。
当然，它更是充斥于我的生活.可惜我玩来玩去，也只是打发走了时间.它对我的生活来说只有debuff…真就是**”Just for fun”**.
这样的Fun它不能持续啊,转眼我就要被生活教训.生活拿鞭子抽我,叫我放下手中的游戏,让我看清眼前，看那些真实存在的东西.

Existentialism

上大学后,高中的那些烦恼可能确实消失了,但我想到,时间、成长这些东西,他不是线性的,不是像游戏通关一样,一关一关的打.生活的拦路虎像梦魇一样在我心里的,考试,生活的收入,买房…这些才是我应该首要考虑的”存在”,我如果实打实的做这些,我感受到的痛苦一定是比每天无目的打游戏少的.痛苦带来反思,反思?

Hexo+Github搭建博客错误(基于第6篇)

发表于 2021-10-18 更新于 2021-10-19

确认一下你的错误发生地点

就是说点一下我！

问题描述(按照出错可能的时间线)

1.hexo d出现异常

出现弹窗让你登录，然后不知道要输入什么乱七八糟的。
我们已经绑定好SSH了，所以不管它。

  • 去根目录，打开_config.yml
  • 拉到底部，找到repository

    type: git repository:git@github.com:08shooter/08shooter.github.io.git #你的仓库地址
    branch: main

  • 去github博客仓库Clone时选择SSH，复制替换到这里就好了。

2.解析失败

我搞了半天怎么打不开网站啊!!!
我们去github仓库点Settings，点击左侧Pages,只要没有显示绿色对钩，就说明网页没有正确连接到域名。
教程里说的是让我们自查ip，反正我是失败了.
解决方法：

可以试试在向你的 DNS 配置解析中多加几条记录(Github的dns)
A @ 185.199.108.153
A @ 185.199.109.153
A @ 185.199.110.153
A @ 185.199.111.153
www CNAME username.github.io.
用你的 Github 用户名替换 username

Tip:如果一切配置没有错误，还显示404的话，建议验证手机版输入你的个人域名，如果
手机版可以显示而电脑版无法显示的话，只需要清除你浏览器的缓存即可。

3.每次hexo d之后都要重新去Pages页面改域名

  • 打开文件拓展名选项(不知道请百度)
  • 在本地根目录创建CNAME.txt
  • 打开输入你的域名
  • 重命名文件，去掉.txt
  • hexo 三步(hexo clean,hexo g,hexo d)

简单学一下马克当(Markdown学习记录)

发表于 2021-10-18

首先首先

注意空格！打完一个符号就要打一个空格捏。

马克当标题语法

============================================
标题前加#和空格，最多加6个，6级最小捏~
马克当不再需要html的

，你输入文字他已经自动补全咯，另外记得不要用空格（spaces）或制表符（ tabs）缩进段落。

怎么换行捏

方法一：在一行的末尾添加两个或多个空格，然后按回车键,即可创建一个换行。
方法二：在行尾添加(
)，请去掉括号食用。

可选语法

还可以在文本下方添加任意数量的 == 号来标识一级标题，或者 – 号来标识二级标题。

那么，我要开始强调了！

首先是粗体(Bold)

前后各添加两个星号（asterisks）或下划线（underscores），建议使用星号（*）捏，因为马克当在如何处理单词或短语中间的下划线上并不一致，然后我粗了。

然后是斜体

和粗体差不多，就是星号数量减1，我斜了。
那怎么同时使用捏?
超级加倍!前后各加三个星号，我粗了，也斜了

简单引用一下~
多个段落的块引用：为段落之间的空白行添加一个 > 符号。
